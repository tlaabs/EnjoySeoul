# 주요 기능
1. 다중 조건 검색  
명칭, 장르, 요금 항목에 대하여 다중 조건 검색이 가능합니다.

2. 찜하기  
마음에 드는 행사를 찜 할 수 있으며, 찜 목록에서 다시 찾아 볼 수 있습니다.
